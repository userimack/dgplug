# dgplug
Scripts for tasks given in #dgplug channel (freenode server)


1. Task: A python script to download image from the url supplied. It should checks if the given url is valid image or if not then it should not be saved.

Script name: check_image.py

Run the script using the following command:
```
Python3 check_image.py <image url>
```

For example:
```
Python3 check_image.py 'https://dgplug.org/assets/img/main-dgp.png'
Output:
Image type detected: png
Success: Image is saved with name: main-dgp.png
```
